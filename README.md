# Schmook Club Files Repository

This is a public repository for hosting large files (audiobooks, ebooks) for the Schmook Club website using Git LFS.

## Files

- **FairyTale.mp3** - Audiobook for the FairyTale book pick

## Usage

Files in this repository are accessed via GitHub's raw URL:
```
https://github.com/Glowstick0017/schmookclub-files/raw/refs/heads/main/[filename]
```

GitHub automatically serves the actual LFS files when accessed through raw URLs.
